## 🚀 主なプロジェクト (Featured Project)

### 🏨 ホテル客室予約・ダッシュボード管理システム (Full-Stack)
> **概要**: 多重の制約条件と厳格なデータの整合性を担保した、ホテル客室予約およびバックオフィス（管理画面）システムです。

* **主な機能**
  * **ユーザー / 管理者権限の分離**: 一般ユーザー向けの客室予約・決済プロセスと、管理者向けの通知（掲示板）管理・客室状態制御機能を実装。
  * **リアルタイムな状態管理**: 空室（`空室`）、清掃中、予約済など、ビジネスルールに基づいた有機的な客室状態の遷移ロジックを設計。
  * **決済および予約情報の連動**: 予約履歴と決済データのマッピング、現地決済・オンライン決済の分岐処理。
* **💡 主な課題解決と成果 (Troubleshooting)**
  * **データの整合性担保**: 外部キー（FK）の依存関係を考慮したデータベーススクリプトの最適化により、参照整合性エラー（`ORA-02449` など）を完全に解決。
  * **正規表現と制約条件の最適化**: DBレベルで正規表現（`REGEXP_LIKE`）や `CHECK` 制約条件を徹底活用し、データ入力の段階から例外をフィルタリング。
